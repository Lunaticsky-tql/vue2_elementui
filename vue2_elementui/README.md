# vue2_elementui

reference: https://zhuanlan.zhihu.com/p/349318513

## How is it built?

```
npm install -g @vue/cli
vue create vue2_elementui
cd vue2_elementui
vue add element
npm install vue-router vue-resource moment axios --S
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```
